# to-do-list

A Todo List made using EJS, HTML, CSS, Node.js, Express.js, and MongoDB is a web application that allows users to create and manage their own personal task lists.

The front-end of the application is built using EJS (Embedded JavaScript) which is a templating engine that allows you to embed JavaScript code in HTML. EJS is used to generate dynamic HTML pages based on data passed to it from the server. HTML and CSS are used to create the overall layout and styling of the application.

The back-end of the application is built using Node.js and Express.js. Node.js is a JavaScript runtime that allows you to run JavaScript on the server side. Express.js is a web framework for Node.js that provides a simple and flexible way to create web applications. It is used to handle incoming HTTP requests and send responses to the client.

For data storage, MongoDB is used as a database management system. MongoDB is a NoSQL database that allows for storing and querying unstructured data in the form of JSON documents. It stores the Todo List items and their statuses.

With this stack, the application allows the user to create new tasks, mark tasks as completed, and delete tasks. The user can also view all the tasks and filter them by completed or pending tasks. The application also provides a way to edit the task's description, and it saves the changes in the MongoDB database.

The application is responsive and can be accessed from any device, and it also allows the user to work offline. This Todo List application is a simple yet effective tool to help users stay organized and manage their tasks efficiently.
